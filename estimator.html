True-Cost Estimator

The True-Cost Estimator is a proof-of-concept design developed from the findings of this research. Rather than focusing only on advertised rent, the estimator combines recurring housing expenses, move-in costs, and qualification information into one simplified interface.

Why it was developed

Participants frequently underestimated the true cost of rental housing because mandatory fees, parking costs, utilities, and deposits were disclosed separately across listing pages.

The True-Cost Estimator demonstrates how these expenses can be consolidated into a single decision-support tool that helps renters compare apartments more accurately before applying.

Design Features

  • Displays the true estimated monthly housing cost instead of advertised rent alone.
  • Summarizes all required recurring fees in one location.
  • Calculates estimated move-in costs before application.
  • Presents qualification requirements using clear, accessible language.
  • Supports easier comparison between rental listings.

Future Implementation

Although presented as a prototype, the True-Cost Estimator could be expanded into a full web application capable of calculating total monthly housing obligations using listing-specific information entered by renters or provided by housing platforms.

The prototype demonstrates one possible human-centered design response to the transparency challenges identified throughout this research.
