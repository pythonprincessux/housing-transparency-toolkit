What this is

This mini-site supports students and early-career renters making high-stakes housing decisions. It focuses on transparency (what you’ll really pay), clarity (what you must qualify for), and accessibility (information that is readable, navigable, and usable for more people).

How to use the toolkit

  1. Start with the Checklist to estimate true monthly cost and your move-in total.
  2. Use the Platform Comparison to see how Zillow and Apartments.com disclose fees and requirements.
  3. Review Accessibility & Plain Language to see how design choices improve clarity.

Honors method: transparency rubric (0–3)

To keep the comparison consistent and repeatable, platforms are scored using a simple rubric: 0 = missing/unclear, 1 = partial, 2 = mostly clear, 3 = clear and easy to find.

This helps move beyond “it feels confusing” into documented findings and actionable recommendations.
